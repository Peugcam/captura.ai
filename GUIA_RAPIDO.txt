1️⃣ INICIE O BACKEND
   └─ Abra terminal e rode: python main_websocket.py
   └─ Deve aparecer: "Uvicorn running on http://0.0.0.0:3000"

2️⃣ ABRA O DASHBOARD
   └─ Abra no navegador: dashboard-tournament.html
   └─ Deve ver: 🟢 Conectado

3️⃣ CONFIGURE TORNEIO
   └─ Clique "⚙️ Configurar Torneio"
   └─ Digite tags dos times (uma por linha)
   └─ Clique "✅ Iniciar Torneio"

  🎯 QUANDO USAR CADA MODO?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

USE MODO AI (🤖) QUANDO:
  • Jogo rolando normalmente
  • IA detectando tudo certinho
  • Você quer só observar

USE MODO CURADOR (🎨) QUANDO:
  • Precisa fazer ajustes manuais
  • Killfeed muito rápido
  • IA errou alguma detecção
  • Quer controle total

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  👀 INDICADORES VISUAIS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

MODO AI ATIVO:
  • Botão: "🎨 Ativar Modo Curador" (ROXO)
  • Sem indicador laranja
  • Tudo automático

MODO CURADOR ATIVO:
  • Botão: "🤖 Ativar Modo AI" (LARANJA)
  • Indicador pulsante: "🎨 MODO CURADOR ATIVO"
  • Você no controle

CONEXÃO:
  • 🟢 Conectado = Tudo OK
  • ⚫ Desconectado = Backend offline

  🐛 PROBLEMAS COMUNS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "Python não reconhecido"
   └─ Tente: py main_websocket.py
   └─ Ou: python3 main_websocket.py

❌ "⚫ Desconectado"
   └─ Certifique que backend está rodando
   └─ Recarregue página (F5)

❌ "Botão Curador não aparece"
   └─ Carregue um torneio primeiro
   └─ Botão só aparece depois de configurar

❌ "Edições não salvam"
   └─ Verifique se está 🟢 Conectado
   └─ Abra console (F12) e veja erros
